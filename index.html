极客工作流（AI+编程）
把每一步写成“需求 → 编码 → 测试”，告别无意义的加班。
SOP
1) 架构设计与 SQL
让 AI 帮你设计表结构、梳理实体关系并生成建表语句。
  • 实体：用户/订单等
  • 关系：一对多/多对多
  • SQL：一键生成建表语句
2) 业务代码与组件生成
提供清晰的 Prompt，让 AI 输出可运行的函数与 UI 组件。
  • 输入：入参定义
  • 输出：期望返回值
  • 边界：异常情况处理
3) 报错排查与优化
把 Error Log 扔给大模型，快速定位根因并获取修复方案。
  • 环境：Node/Python/Go
  • 日志：完整 Error Trace
  • 方案：修复代码与解释
4) 服务器与自动化部署
购买 1bgp 服务器，编写 Dockerfile 与 CI/CD 脚本自动化上线。
  • 环境：配置 Linux 环境
  • 脚本：编写 Shell/CI
  • 上线：一键推送到生产
